# OWASP API Security Top 10 BChecks for Burp Suite

A comprehensive collection of custom **Burp Suite BCheck** scripts designed to detect vulnerabilities listed in the **OWASP API Security Top 10 (2023)**. These scripts automate the identification of critical API security flaws during active scanning with Burp Suite Professional.

---

## 🚀 Overview

BChecks are custom scan definition scripts introduced in Burp Suite Professional that allow users to write custom active scan checks. This repository contains BChecks tailored to automatically probe, identify, and report security issues aligned with the latest OWASP API Security Top 10 standard.

Each script utilizes advanced payloads, detection logic, and WAF-bypass techniques (such as encoding variation, header manipulation, and parameter pollution) to maximize coverage and reliability.

## 🛠️ Installation & Setup

To use these BChecks in Burp Suite Professional, follow these steps:

1. **Download the scripts**: Clone this repository to your local system.
   ```bash
   git clone https://github.com/noname300989/owasp-api-top10-bchecks.git
   cd owasp-api-top10-bchecks
   ```
2. **Import into Burp Suite**:
   - Open **Burp Suite Professional**.
   - Navigate to **Extensions** > **BChecks**.
   - Click the **Import** button.
   - Select the `.bcheck` files you want to import (you can select all of them at once).
3. **Verify scripts**:
   - Burp Suite will validate the syntax of the scripts upon import.
   - Any validation errors will be displayed in the BChecks table. If valid, they will show an `Active` status.

---

## 🔍 How to Run Scans

Once imported and active, these BChecks run automatically whenever you execute an active scan:

1. Capture API requests in Burp Suite (e.g., in target or proxy history).
2. Right-click the request and select **Scan** (or add to a target scope and run a scan).
3. Ensure **Active Scanning** is enabled in your Scan Configuration.
4. Any discovered vulnerabilities will appear in the **Dashboard** / **Target** tab as custom issues generated by BChecks.
